# Soft Roamers Blog

A photo-forward travel portfolio and blog for Soft Roamers.

## Files

- `index.html` - main page content
- `styles.css` - visual design and responsive layout
- `script.js` - interactions, including animated back-to-top scrolling
- `netlify.toml` - Netlify static hosting configuration

## Deployment

This site is designed for GitHub + Netlify:

1. Edit and version the site in a GitHub repository.
2. Connect the repository to Netlify.
3. Netlify publishes the site from the repository root.

No build command is required.
